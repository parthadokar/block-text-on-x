# Privacy Policy – X Word Blocker

X Word Blocker does not collect, store, transmit, or sell any personal data.

## Data Access
The extension temporarily reads website content on x.com in order to identify posts that contain user-defined blocked words. This processing happens entirely locally in the user’s browser.

## Data Storage
The extension stores only the user’s blocked word list using Chrome’s storage API. This data remains on the user’s device and is never transmitted externally.

## Data Sharing
No data is shared with third parties.

## Tracking
The extension does not use analytics, tracking technologies, or remote servers.

## Contact
If you have questions about this privacy policy, you can contact the developer via the Chrome Web Store listing.
